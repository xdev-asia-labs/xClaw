Welcome to the **xClaw** documentation. xClaw is an open-source AI Agent Platform built with TypeScript that lets you build, deploy, and manage AI agents with ease.

## What is xClaw?

xClaw provides a complete toolkit for creating AI-powered applications:

- **Pluggable Skill System** — 11+ pre-built skill packs for programming, healthcare, analytics, and more
- **Plugin Architecture** — Official plugins (ShirtGen.AI, Healthcare) maintained as [git submodules](https://github.com/xdev-asia-labs/xclaw-plugins)
- **Visual Workflow Builder** — Drag-and-drop canvas with 16 node types
- **Multi-LLM Support** — OpenAI, Anthropic Claude, Ollama (local models)
- **Agent Hub Marketplace** — Browse, install, and manage AI agents
- **Multi-Channel** — Telegram, Discord, Web UI, REST API
- **Semantic Memory** — Vector-based long-term memory with cosine similarity
